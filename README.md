# 行气灵眸 · Yin-Yang Moon Phase

一个融合月相、太极、潮汐与手势感应的中文冥想 / 文化体验网页。

> 双手靠近，阴阳相抱；手若相离，潮起潮落。

## ✨ 功能亮点

- 🌙 **月相卡片**：朔月 → 满月 → 晦月，自动适配内容高度
- ☯ **灵性太极图**：暖米白 + 深靛蓝，呼吸式光晕
- 🌊 **潮汐脉冲**：随"阳气比例"自动调节脉动频率
- 🖐 **手势交互**：摄像头识别双掌靠近 → 太极图放大并顺时针加速旋转
- 🎥 **摄像头画面**：右下角实时预览

## 🚀 在线访问

部署后访问：

```
https://xu-12-s.github.io/Yin-Yang-Moon-Phase/
```

> ⚠️ 摄像头权限需要 **HTTPS 或 localhost** 才能获取，请使用上面的部署链接打开。

## 📁 文件

- `行气灵眸(5).html` — 单文件应用（HTML + CSS + JS 全部内联）
- 浏览器直接打开即可运行；推荐使用 Chrome / Edge 最新版

## 🛠 本地预览

```bash
# 任选一种方式启动本地静态服务器
python -m http.server 8000
# 或
npx serve .
```

然后访问 <http://localhost:8000/行气灵眸(5).html>。
